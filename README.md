# ReadyStance

A mobile-first training playbook webapp for visual reaction drills.

## Dot Drill

Configurable drill that flashes a dot at random screen positions in ON/OFF states. Train your observation and reaction by tracking the dot, then analyzing what you saw.

- Adjustable cycles, durations, and countdown via sliders
- Speed control during drill to increase intensity
- Stop/abort to end early and review stats
- Session stats: cycles completed, ON/OFF time, analysis time

## Tech

Vue 3 + Vue Router, built with Vite. No backend — runs entirely in the browser.

## Run

```
npm install
npm run dev
```
